# 🚀 Portfolio

This is my personal portfolio website built with Next.js and GSAP to showcase my projects and journey. The design is inspired by [Tajmirul Islam's portfolio](https://github.com/Tajmirul/portfolio-2.0).

## 🛠 Quick Start

```bash
pnpm i && pnpm dev
```
